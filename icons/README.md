# App Icons

Place your PWA icons here:

- `icon-192.png` — 192×192px (required for Android PWA install)
- `icon-512.png` — 512×512px (required for splash screens and maskable icon)

For a quick placeholder, generate icons at https://favicon.io or use any image editor.
The icon can be a simple 8-ball (black circle with white 8) on a dark background.
